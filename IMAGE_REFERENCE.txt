===============================================
HOW TO USE IN ADMIN PANEL
===============================================

1. Login to Admin Panel: http://localhost:5173/admin
   - Email: admin@suvaya.com
   - Password: admin123

2. Go to the "Menu" tab

3. Click "Add Item" button

4. Fill in the form:
   - Name: Enter product name (e.g., "Chocolate Cake")
   - Description: Enter product description
   - Category: Select from dropdown (Cakes, Pastries, Desserts, etc.)
   - Price: Enter price in KES (e.g., 3500)
   - Image Path: Copy the path from this document (e.g., /Cakes/american-heritage-chocolate-vdx5hPQhXFk-unsplash.jpg)
   - Weight (kg): For cakes, enter weight (e.g., 1.0, 1.5, 2.0) - Leave empty for non-cake items
   - Stock Quantity: Enter 0 for made-to-order items
   - Available: Toggle ON to make item available for purchase

5. Click "Save"

6. The menu item will now appear in the menu with the image!

===============================================
NOTES
===============================================

- All paths start with "/" (e.g., /Menu/... or /Cakes/...)
- Do NOT include "public" in the path
- Image paths are case-sensitive
- Use the exact filename as shown in this document
- If an image doesn't display, check that the filename matches exactly
- You can also add your own images by placing them in the public/Menu or public/Cakes folders
